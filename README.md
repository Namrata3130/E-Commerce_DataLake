# 🚀 Hybrid E-Commerce Lakehouse Data Platform

## 🌍 Overview

This project implements a **cost-optimized hybrid batch + streaming lakehouse architecture** using **AWS S3, Databricks (PySpark + Delta Lake), and Snowflake**.

It simulates a production-grade e-commerce data engineering system capable of:

- Processing batch transactional data 📦  
- Ingesting near real-time clickstream events ⚡  
- Implementing Medallion Architecture (Bronze / Silver / Gold) 🥉🥈🥇  
- Serving curated analytical datasets to Snowflake 📊  

The architecture is cloud-native while intelligently simulating streaming locally to minimize infrastructure cost.

---
## 🧰 Tech Stack

- **Cloud Storage:** AWS S3  
- **Processing Engine:** Databricks Community Edition  
- **Framework:** PySpark  
- **Storage Layer:** Delta Lake  
- **Streaming:** Structured Streaming (locally simulated events)  
- **Data Warehouse:** Snowflake  
- **Language:** Python  
- **Data Formats:** CSV, JSON, Delta
